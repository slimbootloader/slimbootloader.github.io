Stitch IFWI Image

Intel platforms use Intel FirmWare Image (IFWI) which in addition to SBL firmware includes other firmware components to boot.

SBL image occupies BIOS region in the IFWI image.

See Supported Hardware on stitching instructions for supported platforms.
